# Psychology Booking System

Psychology Booking System là một ứng dụng web hỗ trợ đặt lịch tư vấn tâm lý, giúp kết nối khách hàng với các chuyên gia tâm lý một cách thuận tiện và hiệu quả. Dự án được xây dựng với mục tiêu mô phỏng một hệ thống đặt lịch thực tế, tập trung vào quản lý người dùng, lịch hẹn và quy trình tư vấn.

## Mục tiêu dự án

- Xây dựng hệ thống đặt lịch tư vấn tâm lý trực tuyến
- Áp dụng kiến thức về lập trình web và thiết kế hệ thống
- Thực hành phân tích nghiệp vụ, phân quyền và quản lý dữ liệu
- Làm quen với quy trình phát triển phần mềm thực tế

## Chức năng chính

- Đăng ký và đăng nhập người dùng
- Phân quyền cơ bản (Admin / psychologist / Client)
- Đặt lịch tư vấn tâm lý
- Kiểm tra nhanh về tâm lý qua các bài test
- Xem lịch sử các buổi tư vấn
- Xem các bài đăng và podcast
- chat real-time 

## Công nghệ sử dụng

- Backend: .NET (C#)
- Framework: ASP.NET Core MVC
- Frontend: HTML, CSS, JavaScript
- Cơ sở dữ liệu: SQL Server
- Công cụ phát triển: Visual Studio / Visual Studio Code.
- Quản lý mã nguồn: GitHub
